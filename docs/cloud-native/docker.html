# MySQL

docker安装 mysql ，并映射数据到本地

docker run -p 3306:3306 --name kop -v /data/mysql/kop:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=kop@2022 -d mysql:8.0.21 --character-set-server=utf8mb4 --collation-server=utf8mb4_unicode_ci
